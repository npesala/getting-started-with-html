OVERVIEW:

The basics of web:

-The Web
-HTML
-URLS
-HTTP
-Web Apps

Browsers: makes requests via the internet to servers. These requests are using http and servers respond with files that your browser displays Ex: firefox, chrome
Servers: computers similiar to are but they host files and are not placed on desks
Http:the main protocol that the browser uses to communicate with the servers

What does HTML stand for? Hyper Text Markup Language It is the basis of almost every webpage and glues everything together.

It is made up of:

-text context "what you see"
-markup "what it looks like"
-References to other docs ex: images & videos
-links to other pages

TAGS & ELEMENTS

HTML documents are made up of element created by tags.

Types of tags:
-b-tag: Bold
-em-tag: emphasis (italics)
-img-tag:images
-br-tag:break (forces HTML to produce multiple lines)
-p-tag:paragraph (creates invisible box around text)

What is the difference between inline and block?
Inline ends the current line. Ex: br-tag Block creates an invisible box around the text. Ex: p-tag
